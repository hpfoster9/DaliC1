Projects

Here are some projects I've been working on since high school! While I probably should have been doing homework instead of coding, I hope you will find some entertainment from them!

The Ghost Game

For my High School Graduation project I spent 6 weeks developing this Java Servlet, JSP, and JQuery based word game. The game uses md5 hashing to generate unique game ids and basic bootstrap to style the page. I haven't take an HTML or CSS class and so try not to be too tough on my high school css skills :p Anywho, if you would like to play, my website is active here (also I'm hosting with Heroku for free so it may take ~15 seconds to load initially)


Sierpinski Pyramid

During the fall term, while I was supposed to be studying for my Ling 1 exam, I decided it would be a perfect time to learn Processing 3 and build some fractals! Below is my first creation: a spinoff of the classic Sierpinski triangle... but now as a pyramid!


Dragon Curve

Furthering the previous Processing project, I decided to try to make a 3D version of the classic 'Dragon Curve' fractal. I have created a function that generates a curve based on the degrees rotated around the vertical axis. Additional fractals make the project a bit messy and so I have used a 90 degree interval between fractals.
